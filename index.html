QTimeLapse is a Qt/GPhoto-based program that can use an external camera to take repeated pictures at certain intervals with the intention of stringing them together into a timelapse movie. It is based heavily on Tim Nugent's GTimeLapse.

The goals of QTimeLapse are to create a simple, flexible time-lapse program that is both easy to use and easy to contribute to. GTimeLapse was a great program, but development has stalled, as has the previous iteration of a Qt time-lapse program (https://launchpad.net/qtimelapse). To this end, this is a Qt Creator project that can be build upon by anyone, making it easy for developers to add features and for users to install.
